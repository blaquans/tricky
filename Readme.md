
## Tricky

***Tricky*** is an R package with miscellaneous R functions that are useful to me.

## Functions

### Formatting numbers

* `french_formatting()`
* `percent_formatting()`

### Unformatting numbers

* `unfrench_formatting()`

### ggplot themes

* `theme_elegant()` : a simple theme for graphics
* `theme_cartography()` : a theme for maps
